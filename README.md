# Loan Application Statistical Analysis Project

This project performs statistical and probability analysis on a loan application dataset using Python.
The project includes descriptive statistics, probability concepts, data visualization, and basic linear algebra operations.

---

# Dataset Information

The dataset contains around 5000 customer loan records with the following columns:

* Customer_ID
* Age
* Income
* Loan_Amount
* Credit_Score
* Loan_Term
* Default_Status

# Key Insights

* Customers with low credit scores have higher default probability.
* Loan amount data contains high variation.
* Income and loan amount are positively related.
* Statistical analysis helps identify risky customers.
